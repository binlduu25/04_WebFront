input 태그

text 관련 input 태그

type = "text"

- 한 줄짜리 문자열(텍스트)를 입력할 수 있는 입력상자
- input 태그 타입의 기본값으로, 생략 가능
- inline 형식이다
    
    
text 관련 input 태그 공용 속성

- size : 입력상자 크기
- maxlength : 입력 받는 텍스트의 최대 길이
- placeholder : 입력상자에 작성될 내용 설명
