form 태그

-div와 같은 영역(block)
내부에 작성된 input 태그 값을 서버로 전달 역할

[속성]
1) action : 내부에 작성된 값을 전달할 서버 또는 페이지 주소 작성        
            미작성 시 기본값으로 현재 페이지의 url 사용

2) name : form 태그의 이름을 지정하여 form 태그 구분 (js 사용)

3) method(방식) : 서버로 데이터를 전달하는 방식을 지정하는 속성
                - get / post 방식 지정 가능(기본값 get)

4) target : action에 지정된 주소를 현재탭/새탭 어디서 열지 지정
