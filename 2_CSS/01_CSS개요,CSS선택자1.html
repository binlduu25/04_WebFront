CSS(Cascading Style Sheets)

마크업언어(html)가 실제로 화면에 표시되는 방법(모양, 색상, 크기, 스타일..)을
기술하는 언어
즉, html 코드가 화면에 어떤 스타일로 출력될지 지정하는 것

CSS Selector(선택자)

선택자란?
현재 html 문서 내에서 특정한 요소를 선택하는 문자 또는 문자열. 
(*요소 : 태그로 만들어진 것)
-> 스타일을 적용하고자 하는 요소를 선택하는 '방법' : 선택자

태그 선택자

현재 HTML 문서 내에 같은 태그를 선택하는 선택자.

[작성법]

태그명{
    CSS 코드;
}
strong 태그에 작성된 내용
span 태그에 작성된 내용
  • strong 태그에 작성된 내용
  • span 태그에 작성된 내용

id 선택자 

    HTML 문서 내에서 id 속성값이 일치하는 요소를 선택하는 선택자
    * id 속성 : 고유한 식별자로서의 역할을 하는 속성
        - 하나의 HTML 문서 내에서 같은 id 중복으로 작성 금지

    [작성법]

    #id속성값{ * '#' = id
        CSS코드;
    }

class 선택자


    html 문서 내에서 같은 class 속성값을 가지는 태그를 모두 선택하는 선택자

    [작성법]
    .class 속성값{ * '.'= class
        css 코드;
    }
